# fastcache

High-performance in-memory caching library with LRU/LFU eviction

## Installation

```bash
pip install fastcache
```

## Quick Start

```python
from fastcache import Fastcache
client = Fastcache()
client.run()
```

## Features

- High performance and memory efficient
- Comprehensive error handling
- Well tested with full coverage
- MIT licensed
